# African Science Media House — Competitor Benchmarking Dashboard

Power BI dashboard rebuilding a competitor benchmarking analysis originally 
developed for an industrial/academic project with My Science Journey 
(UK-based science communication organisation), in partnership with the 
University of Liverpool.

## Business context
My Science Journey is exploring a proposed "African Science Media House" — 
a multimedia platform to increase visibility of African scientific research. 
This dashboard benchmarks 7 comparable science communication organisations 
against My Science Journey's proposed platform, to identify strategic 
positioning gaps.

## What this dashboard shows
- **Benchmark score comparison** — ranks all 7 organisations across 5 
  weighted criteria (multimedia capability, Africa focus, audience reach, 
  funding sustainability, content quality)
- **Competitor positioning map** — plots multimedia capability against 
  Africa-specific focus, revealing an uncontested market quadrant
- **Content format gap analysis** — heatmap showing which of 10 content 
  formats (podcast, video, newsletter, etc.) each competitor offers

## Key insight
No existing competitor combines full multimedia capability with strong 
Africa-specific focus — My Science Journey's proposed platform is 
positioned to fill that gap.

## Tools and techniques
- Power BI Desktop
- DAX measures: `RANKX`, `CALCULATE`, `AVERAGE` for dynamic competitive ranking
- Conditional formatting for heatmap visualisation
- Interactive slicers for cross-filtering
